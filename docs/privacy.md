# Privacy

Meshtrack Studio v1 collects no analytics.

## What Stays Local

- Project snapshots saved in IndexedDB.
- Imported and exported JSON project files.
- Browser audio state.

## Collaboration

When a user joins a room, Yjs updates are sent to peers through WebRTC data channels. Public `y-webrtc` signaling servers are used for peer discovery. They do not act as the project sync server.

Room names should be treated as share links. Anyone with the same room name can attempt to join while the room is active.
