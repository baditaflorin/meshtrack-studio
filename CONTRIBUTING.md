# Contributing

Thanks for helping improve Meshtrack Studio.

## Local setup

1. Install Node.js 22 or newer.
2. Run `npm install`.
3. Run `make install-hooks`.
4. Run `make dev`.

## Commit style

Use Conventional Commits:

- `feat:` for user-facing features
- `fix:` for bug fixes
- `docs:` for documentation
- `test:` for tests
- `chore:` for maintenance
- `ops:` for operational scripts

Run `make lint test build smoke` before pushing.
